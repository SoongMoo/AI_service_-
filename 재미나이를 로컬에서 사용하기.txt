재미나이를 로컬에서 사용하기
1. https://ollama.com/download에서 OllamaSetup.exe 파일을 다운 받아 설치를 한다.

2.  쉘에서 ollama run gemma3:4b 를 실행한다.
> ollama run gemma3:4b 
3. 실행이 되면 테스트를 해본다,
>>> 안녕하세요.

4. 주피터에서 실행해보자.
!pip install ollama


from ollama import chat
response = chat(
    model="gemma3:4b",
    messages=[
        {"role":"user","content":"대한민국 수도는?"}
    ]
)

print(response["message"]["content"])
